Rachel Conforti - Project: Local Search NQueens

Algorithm: Min-Conflict

How to run the program: run in any IDE that runs python, press the run the button and the code will run itself. 

Want to change the number of queens for the boards?
  - change line 158 from 8 to the desired number 

Want to change how many boards are run?
  - change line 154 from 50 to the desired number
